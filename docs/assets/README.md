# Demo GIF

`pdf-snake-demo.gif` is intentionally not committed yet.

To create it:

1. Open `generated/playable-pdf-snake.pdf` in Chrome desktop.
2. Use ScreenToGif, OBS, or another screen recorder.
3. Record 8 to 10 seconds: click START, steer the snake, eat food, hit a wall, show the death blink, and show GAME / OVER.
4. Save the result as `docs/assets/pdf-snake-demo.gif`.
5. Replace the README's "Demo GIF coming soon" text with:

```md
![PDF Snake demo](docs/assets/pdf-snake-demo.gif)
```
