# v0.5.0 - Playable PDF Snake

First public release of PDF Snake.

Open `playable-pdf-snake.pdf` in Chrome desktop PDF viewer and click START.

Features:
- Playable Snake inside a PDF file
- AcroForm fields + embedded JavaScript
- Static 20×20 grid with text overlay rendering
- @ / O / X ASCII renderer
- Death animation and GAME OVER screen

Recommended environment:
- Chrome desktop PDF viewer

Notes:
- Mobile PDF viewers and in-app PDF previews may not run the game.
